Project Title

Jobs Portal

Introduction

The project title is "Jobs Portal".It is an Online Job Search Agent. The Jobs are searched based on the Location given by the User.In this the User needs to enter his/her Details and the Location where he/she requires a Job. The Companies nearby the Location are Displayed and the user needs to Cross-Check his/her Qualifications with the Company's Qualification, If it matches then he/she can give their Personal Details.
Objective

1.Easy to Search Jobs.
2.No Transportation Issues.
3.No need to visit the Company for Application Filling.
4.Easy Interaction with Comapany and User.
Languages Used

This project as title "Jobs Portal" is comes under the Database Management System (DBMS). This application is developed with the help of HTML, CSS and SQL. This application can also be run on the network environment so it can be said as network application
Tools/Platform

This project is developed using the tools, which are most suited for development of the Application Package. These tools are as follows: - 1.PHP (For front end)
2.MySQL (For Database Storage as Back end)
